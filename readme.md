## A highly experimental popcorn-time client build with:

* ES6
* React 15.0
* Redix
* Sass
* ImmutableJS
* Electron 1.0.0
* Webpack

## Installation:
```
git clone https://github.com/amilajack/popcorn-desktop-experimental.git
cd popcorn-desktop-experimental

npm i -g gulp
npm i

gulp
npm run dev
```
